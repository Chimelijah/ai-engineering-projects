# Local RAG API with FastAPI & Ollama

## 🛠️ Tech Stack
**Technologies:** `Python, FastAPI, Ollama, ChromaDB, TinyLlama`

## 📖 Project Overview
Built a zero-cost local RAG API querying TinyLlama via Ollama and storing embeddings in ChromaDB.

## 🚀 Key Learnings
* Configuration and deployment of `Python, FastAPI, Ollama, ChromaDB, TinyLlama` in a production-like environment.
* Deep-dive analysis of system logs and network traffic.
* Documentation of technical workflows for architectural clarity.
